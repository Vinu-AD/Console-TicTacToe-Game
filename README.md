# Tic Tac Toe – Java Console Game

A simple **Tic Tac Toe (XO)** game built using **Java**, designed for **two players** in the **command line**. This game supports custom board sizes and includes input validation.

---

## Features

- Two-player mode
- Console-based interface
- Custom board dimensions (3x3, 4x4, etc.)
- Input validation and turn-based logic
- Winner and draw detection

---

## Concepts used

- Loops, Arrays, Conditional Logic
- Scanner class (for player input)

---
